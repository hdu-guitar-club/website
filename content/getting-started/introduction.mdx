# 介绍

欢迎使用 **HDJX Docs** 文档系统！这是一个基于 Nextra 构建的现代化文档站点。

## 特性

- 🚀 **快速** - 基于 Next.js 的静态生成，加载速度极快
- 🎨 **美观** - 使用 Tailwind CSS 构建，支持深色模式
- 📱 **响应式** - 完美适配各种设备尺寸
- 🔍 **可搜索** - 内置全文搜索功能
- 🌐 **国际化** - 支持多语言文档

## 技术栈

- [Next.js](https://nextjs.org/) - React 框架
- [Nextra](https://nextra.site/) - 文档生成器
- [Tailwind CSS](https://tailwindcss.com/) - CSS 框架
- [TypeScript](https://www.typescriptlang.org/) - 类型安全
