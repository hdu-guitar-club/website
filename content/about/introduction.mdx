# 关于杭电吉协

## 核心概况

| 项目 | 内容 |
|------|------|
| 全称 | 杭州电子科技大学吉他协会 |
| 简称 | 杭电吉协 |
| 成立时间 | 2009年 |
| 固定地址 | 学生活动中心南楼A105（学活南A105） |
| 社团资质 | 校十佳社团之一 |

## 核心定位

杭电吉协是以**乐队为核心**，以**乐队四大件（吉他、贝斯、键盘、鼓）＋主唱**为基础的现场型音乐社团。

## 社团特色

- **包容性强**：欢迎各种乐器加入，包容各类音乐曲风
- **现场导向**：注重舞台表演和现场演出经验
- **开放合作**：与各大高校开展合作，提供跨校舞台表演机会

## 活动举办场地

- 五四广场
- 东操场
- 学活剧院
- 藤廊
- 其他校内场地
