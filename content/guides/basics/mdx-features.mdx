# MDX 特性

探索 MDX 的强大功能，结合 Markdown 和 JSX。

## 什么是 MDX？

MDX 是 Markdown 的超集，允许你在 Markdown 中使用 JSX 组件。

## 基本语法

### Markdown 语法

```mdx
# 标题

这是一段**加粗**文本和*斜体*文本。

- 列表项 1
- 列表项 2
- 列表项 3

[链接文本](https://example.com)
```

### 使用 JSX 组件

```mdx
import { Button } from '@/components/ui/button';
import { Alert } from '@/components/ui/alert';

<Alert>
  这是一个提示框
</Alert>

<Button>点击我</Button>
```

## 代码块

### 基本代码块

````mdx
```javascript
const greeting = 'Hello, World!';
console.log(greeting);
```
````

### 带文件名

````mdx
```javascript title="utils.js"
export const formatDate = (date) => {
  return new Date(date).toLocaleDateString();
};
```
````

### 高亮行

````mdx
```javascript {2-3} showLineNumbers
function greet(name) {
  console.log(`Hello, ${name}!`);  // 高亮
  return `Welcome, ${name}!`;       // 高亮
}
```
````

## 自定义组件

在 `mdx-components.tsx` 中注册自定义组件：

```typescript
import { Button } from '@/components/ui/button';

export const useMDXComponents = (components) => {
  return {
    ...components,
    Button,
  };
};
```

然后在 MDX 中直接使用：

```mdx
<Button variant="primary">自定义按钮</Button>
```

## Frontmatter

在文件顶部添加元数据：

```mdx
---
title: 页面标题
description: 页面描述
---

# 内容开始
```
