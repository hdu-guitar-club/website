# 路由配置

了解如何配置 Nextra 文档站点的路由。

## 基本路由

Nextra 使用文件系统路由，`content/` 目录下的文件会自动映射到对应的 URL。

### 文件到 URL 映射

| 文件路径 | URL |
|---------|-----|
| `content/index.mdx` | `/docs` |
| `content/getting-started.mdx` | `/docs/getting-started` |
| `content/guides/basics/routing.mdx` | `/docs/guides/basics/routing` |

## _meta.ts 配置

`_meta.ts` 文件用于定义侧边栏导航结构：

```typescript
export default {
  index: {
    title: '首页',
  },
  'getting-started': {
    title: '快速开始',
  },
};
```

## 导航类型

### 页面类型

```typescript
{
  'page-name': {
    type: 'page',
    title: '页面标题',
    href: '/custom-url', // 可选，外部链接
  }
}
```

### 菜单类型

```typescript
{
  'menu-name': {
    type: 'menu',
    title: '菜单标题',
  }
}
```

### 分隔符

```typescript
{
  'separator-name': {
    type: 'separator',
    title: '分组标题',
  }
}
```

## 隐藏页面

```typescript
{
  'hidden-page': {
    display: 'hidden',
  }
}
```

## 排序

默认按字母顺序排序，可以通过文件名前缀控制：

```
01-introduction.mdx
02-installation.mdx
03-configuration.mdx
```
