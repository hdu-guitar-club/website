# 国际化

为文档站点添加多语言支持。

## 配置国际化

### Next.js 配置

```javascript
// next.config.js
const withNextra = require('nextra')({
  theme: 'nextra-theme-docs',
  themeConfig: './theme.config.tsx',
  defaultLocale: 'zh',
  locales: ['zh', 'en'],
});

module.exports = withNextra();
```

## 多语言内容

### 中文内容

```mdx
// content/zh/getting-started/introduction.mdx
---
title: 介绍
---

# 介绍

欢迎使用 HDJX Docs 文档系统！
```

### 英文内容

```mdx
// content/en/getting-started/introduction.mdx
---
title: Introduction
---

# Introduction

Welcome to HDJX Docs documentation system!
```

## 语言切换

Nextra 会自动在导航栏添加语言切换器，用户可以轻松切换语言。

## 翻译最佳实践

1. **保持同步**：确保所有语言版本的内容结构一致
2. **使用翻译键**：对于重复使用的文本，使用统一的翻译键
3. **本地化日期和数字**：使用适当的格式化工具
4. **考虑文化差异**：注意颜色、图标等的文化含义
