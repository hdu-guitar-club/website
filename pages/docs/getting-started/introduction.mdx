# 介绍

HDJX Docs 是一个现代化的文档站点解决方案。

## 核心特性

### MDX 支持

在 Markdown 中直接使用 React 组件：

```jsx
import { useState } from 'react'

export function Counter() {
  const [count, setCount] = useState(0)
  return (
    <button onClick={() => setCount(count + 1)}>
      点击次数: {count}
    </button>
  )
}

<Counter />
```

### 代码高亮

支持多种编程语言的语法高亮：

```typescript
interface IProps {
  title: string
  description?: string
}

export const Component = (props: IProps) => {
  const { title, description } = props
  return (
    <div>
      <h1>{title}</h1>
      {description && <p>{description}</p>}
    </div>
  )
}
```

### 表格支持

| 特性 | 描述 |
| --- | --- |
| MDX | 在 Markdown 中使用 React |
| 搜索 | 内置全文搜索功能 |
| 主题 | 支持深色/浅色模式 |
