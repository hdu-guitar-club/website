# 安装

本指南将帮助你快速安装和配置 HDJX Docs。

## 系统要求

- Node.js 18.17 或更高版本
- npm、yarn 或 pnpm

## 创建项目

### 方式一：克隆模板

```bash
git clone https://github.com/your-repo/hdjx-docs.git
cd hdjx-docs
npm install
```

### 方式二：手动创建

```bash
# 创建项目目录
mkdir hdjx-docs
cd hdjx-docs

# 初始化项目
npm init -y

# 安装依赖
npm install next nextra nextra-theme-docs react react-dom
npm install -D typescript @types/node @types/react @types/react-dom
```

## 项目结构

```
hdjx-docs/
├── pages/
│   ├── index.mdx        # 首页
│   ├── _meta.json       # 顶部导航配置
│   └── docs/            # 文档目录
│       ├── index.mdx
│       ├── _meta.json   # 侧边栏配置
│       └── ...
├── theme.config.tsx     # 主题配置
├── next.config.js       # Next.js 配置
└── package.json
```

## 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000 查看你的文档站点。
