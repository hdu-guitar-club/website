# 基础用法

学习如何使用 HDJX Docs 编写文档。

## 编写文档

所有文档文件都放在 `pages` 目录下，使用 `.mdx` 或 `.md` 格式。

### 创建新页面

在 `pages/docs` 目录下创建新的 `.mdx` 文件：

```markdown
# 页面标题

这是页面内容...
```

### 配置导航

在 `_meta.json` 中配置侧边栏导航：

```json
{
  "introduction": "介绍",
  "installation": "安装",
  "guide": {
    "title": "指南",
    "type": "separator"
  },
  "basic-usage": "基础用法"
}
```

## Markdown 扩展

### 代码块

支持语法高亮和行号：

```tsx showLineNumbers
export const hello = () => {
  console.log('Hello, World!')
}
```
