# 概述

欢迎使用 HDJX Docs！这是一个基于 [Nextra](https://nextra.site/) 构建的文档站点模板。

## 什么是 HDJX Docs？

HDJX Docs 是一个开箱即用的文档解决方案，帮助你快速搭建专业的文档站点。

## 为什么选择 Nextra？

Nextra 是一个基于 Next.js 的文档框架，它提供了：

- **简洁的配置** - 只需少量配置即可开始
- **强大的功能** - 支持全文搜索、深色模式等
- **优秀的性能** - 基于 Next.js 的静态生成
- **MDX 支持** - 在 Markdown 中使用 React 组件
