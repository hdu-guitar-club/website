# 自定义主题

学习如何深度自定义 HDJX Docs 的样式和主题。

## 主题颜色

在 `theme.config.tsx` 中设置主色调：

```tsx
export default {
  primaryHue: 211,
  primarySaturation: 100,
}
```

## 自定义样式

### 全局样式

创建 `styles/globals.css` 文件：

```css
:root {
  --nextra-primary-hue: 211deg;
}

.dark {
  --nextra-primary-hue: 200deg;
}

/* 自定义代码块样式 */
pre {
  border-radius: 8px;
}

/* 自定义链接样式 */
a {
  text-decoration: none;
}
```

### 组件样式

在 MDX 中使用内联样式或 CSS 类：

```jsx
<div style={{ 
  padding: '1rem', 
  background: '#f5f5f5',
  borderRadius: '8px' 
}}>
  自定义样式的内容
</div>
```

## 自定义组件

### 创建自定义组件

```tsx
import { useState } from 'react'

interface ICounterProps {
  initialCount?: number
}

export const Counter = (props: ICounterProps) => {
  const { initialCount = 0 } = props
  const [count, setCount] = useState(initialCount)
  
  return (
    <button 
      onClick={() => setCount(c => c + 1)}
      style={{
        padding: '0.5rem 1rem',
        background: '#0070f3',
        color: 'white',
        border: 'none',
        borderRadius: '4px',
        cursor: 'pointer'
      }}
    >
      Count: {count}
    </button>
  )
}
```

### 在 MDX 中使用

```mdx
import { Counter } from '../components/Counter'

<Counter initialCount={10} />
```
