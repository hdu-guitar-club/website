# 活动一览

杭电吉协每年举办丰富多彩的音乐活动，为成员提供充足的舞台表演机会。

## 年度活动清单

| 活动名称 | 时间/频次 | 说明 |
|----------|-----------|------|
| 百团招新 | 每年9月 | 新成员加入的主要渠道 |
| 各学院迎新晚会 | 每年9月起，每学期1-2次 | 为各学院迎新演出提供乐队表演 |
| 藤廊音乐会 | 第一学期举办，每学期1-2次 | 校园户外音乐活动 |
| 草坪音乐会 | 每学期不定期举办 | 轻松休闲的音乐聚会 |
| 跨年音乐节/小GR | 每年12月底举办 | 年末音乐盛典 |
| 社团巡礼节 | 每年5月底举办 | 校园社团展示活动 |
| 校十佳歌手总决赛 | 每年5月底举办 | 吉协负责现场伴奏 |
| **GR音乐节** | 每年5月底或6月初举办 | **年度核心大型演出** |
| 与各大高校合作舞台 | 全年不定期举办 | 跨校交流演出 |

## GR音乐节

GR音乐节是杭电吉协年度最重要的核心大型演出活动，每年5月底或6月初举办。这是吉协成员展示一年练习成果的重要舞台，也是杭电校园音乐文化的年度盛事。

### 活动亮点

- 多支乐队轮番登台表演
- 多种音乐风格同台竞技
- 专业舞台设备与灯光
- 全校师生共同参与

## 藤廊音乐会

藤廊音乐会是吉协的特色户外音乐活动，在杭电下沙校区生活区藤廊举办。以下以2024年3月29日场次为参考模板，介绍全流程：

### 活动流程

#### 1. 前期筹备

- 微信公众号发布预热推文
- 评论区设为点歌区域，观众留言想听歌曲
- 随机抽取幸运观众的点歌加入歌单
- 完成推文、海报制作等各类前期宣传工作

#### 2. 中期筹备

- 根据点歌结果+乐队选曲，基本确定最终歌单
- 基本确定各表演队伍的演出顺序

#### 3. 后期筹备

- 各表演队伍自行开展排练
- 组织人员将现场需要的各类乐器及设备搬运至藤廊场地

#### 4. 现场筹备

- 完成舞台搭建、设备调试
- 工作人员到位，做好演出前各项准备
- 演员上台展示彩排

#### 5. 正式演出

- 各队伍按既定顺序上台表演
- 全程安排人员拍照片、录视频

#### 6. 收尾阶段

- 演出结束后，将所有乐器、设备搬回学活南A105排练房
- 整理藤廊现场卫生
- 运营各官方账号，将拍摄的照片和视频编辑后发布

## 跨校合作

吉协积极与各大高校开展合作，为成员提供跨校舞台表演机会，拓展音乐视野，结识更多音乐爱好者。
